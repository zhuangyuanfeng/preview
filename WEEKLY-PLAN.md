## 🧪 每周日裸写检测

> 每周日固定 40-60min，模拟面试手写环境
> 规则：空白编辑器，不查资料，不用 AI，限时完成

### 检测方式
1. 从本周做过的算法题中随机挑 2 题，每题限时 15min
2. 从已学过的手写题中挑 1 题，限时 10min
3. 写完后自己检查，记录卡壳的地方

### 评分标准（自评）
- ⭐⭐⭐⭐⭐ 一遍过，思路清晰
- ⭐⭐⭐⭐ 小 bug 但能自己改对
- ⭐⭐⭐ 思路对但写不完整
- ⭐⭐ 思路模糊，靠记忆拼凑
- ⭐ 完全写不出来 → 下周必须重新学这个知识点

## 什么适合碎片时间看

✅ 适合：
- 面经文章（掘金/公众号）
- Vue3 源码解读文章（先看文章建立概念，晚上再看源码）
- React 概念科普（Fiber/Hooks 原理文章）
- 浏览器/网络/HTTP 八股文
- 系统设计案例分析
- AI 技术博客（LLM/RAG/Agent 概念）
- LeetCode 题解（先看思路，晚上再写）

❌ 不适合：
- 写代码（碎片时间写不了完整思路）
- 算法动手做（需要专注）
- 源码 debug 跟踪

## 砍掉的内容

| 内容 | 理由 |
|------|------|
| CSS 布局与动画 | 8年经验够用 |
| TCP 细节 | 性价比低 |
