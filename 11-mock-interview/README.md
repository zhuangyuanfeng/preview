# 模拟面试

> 优先级：⭐⭐⭐⭐ | 建议时间：第11-12周

## 面试流程准备

### 自我介绍（1分钟版 + 3分钟版）
- [ ] 突出：8年经验 + 电商深耕 + AI方向转型
- [ ] 准备中文和英文版本

### 技术面常见问题分类
- [ ] JavaScript 核心（事件循环、闭包、原型链）
- [ ] React/Vue 框架深度（源码级）
- [ ] 工程化（Webpack/Vite、CI/CD）
- [ ] 网络与安全
- [ ] 算法题（手写）
- [ ] 系统设计题
- [ ] AI 相关知识

### 行为面试（HR面/终面）
- [ ] 为什么离职？（准备正面回答）
- [ ] 你最有成就感的项目？
- [ ] 遇到的最大技术挑战？
- [ ] 如何看待技术选型？
- [ ] 团队协作中的冲突处理
- [ ] 对未来的职业规划？

## 模拟练习
- [ ] 找朋友或用 AI 做模拟面试
- [ ] 录制自己的回答，回看改进
- [ ] 整理常见追问及应对

## 公司研究
- [ ] 目标公司清单（按优先级）
- [ ] 每家公司的技术栈调研
- [ ] 面试流程了解（几轮、侧重点）

## 心态管理
- 前几场面试当练手，不要对结果有太高期望
- 每次面试后复盘，记录问到的题目
- 保持学习节奏，不因拿到 offer 就松懈
