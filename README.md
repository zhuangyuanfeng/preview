---

## 🧭 v2 与 v1 的区别（为什么要重整）

| 维度 | v1（旧） | v2（新） | 为什么改 |
|---|---|---|---|
| 节奏 | 每日打卡 | **周里程碑** | 心流型，深夜启动困难，日打卡破心流 |
| AI 使用 | 学习时严禁 | **AI 学，自己写** | 视觉/讨论型偏好，禁 AI 等于砍输入方式 |
| 模块呈现 | 一长串 checkbox | **图 + Demo + 话术** | 视觉先导，对纯文字 todo 抗拒 |
| 时段 | 强制 23:00-00:30 | **任意心流时段** | 心流难启动，深夜成本最高 |
| 验收 | 周日裸写检测 | **5 分钟"自己讲一遍"** | 记忆力受限，裸写压力过大 |
| 兴趣驱动 | 无 | **每周 1 个工作痛点** | Q12 没选兴趣型，需用"立即有用"补 |

## 📐 每个模块的统一结构

进入任何一个模块 README，都会看到这 5 个区块：

```
🗺️ 模块全景图       —— 视觉先导，先看这个
🎯 核心概念清单     —— 必须能"用自己的话说清楚"
💪 必出 Demo 清单   —— 防听懂幻觉，每个 50 行内
🎤 面试话术（STAR）—— 直接抄进简历/面试
🔧 工作痛点联动     —— 这个模块帮你解决工作中哪些事
```

**核心原则**：不画图、不出 demo 的知识点 = 没学。
