# 算法与数据结构

> 聚焦 LeetCode Hot 100，贯穿12周，每天1-2题

## 刷题节奏

- 工作日每天 1 题（早上或午休，保持手感）
- 周末可以多刷 2-3 题补进度
- 3天后复习，1周后再复习（间隔重复）
- 每题先想15分钟，没思路就看题解，**理解思路后自己重写**
