#15 三数之和 Medium

Hot 100 · 双指针 · 第2周

图示

题目

给定一个整数数组 nums，找出所有满足 a + b + c = 0 的三元组 [a, b, c]，答案中不能包含重复的三元组。

输入：nums = [-1, 0, 1, 2, -1, -4]
输出：[[-1, -1, 2], [-1, 0, 1]]

先想一想

暴力三层循环是 O(n³)，怎么优化？

排序 + 双指针：

先排序 O(n log n)，然后遍历数组，固定一个数 nums[i]，对剩余部分用对撞双指针找两个数使三数之和为 0。

外层循环 O(n) × 内层双指针 O(n) = 总共 O(n²)，比暴力快了一个数量级。

核心思路

排序后遍历 i，对于每个 nums[i]，用对撞双指针在 i+1 到末尾找两个数，使三数之和为 0。
本质：把三数之和问题降维成对每个 i 做一次两数之和。

解题步骤（逐步展开）

点击每一步展开详细说明

1 排序数组 ▶

nums.sort((a, b) => a - b)

排序是整个算法的基础：让双指针可以根据 sum 的大小移动方向，同时让相同元素相邻，方便去重。

例：[-1, 0, 1, 2, -1, -4] → [-4, -1, -1, 0, 1, 2]

2 遍历 i 从 0 到 n-3，固定 nums[i] ▶

外层循环固定第一个数 nums[i]，问题变成：在 i+1 到 n-1 范围内找两个数之和等于 -nums[i]。

剪枝优化：如果 nums[i] > 0，后面的数都比它大，三数之和不可能为 0，直接 break。

3 left = i+1, right = n-1，双指针找 target = -nums[i] ▶

在已排序的子数组中，左指针从 i+1 开始，右指针从末尾开始，向中间逼近。

计算 sum = nums[i] + nums[left] + nums[right]：

  • sum < 0 → 和太小，left++ 让和变大
  • sum > 0 → 和太大，right-- 让和变小
  • sum === 0 → 找到一组答案！
4 sum === 0 → 记录答案并跳过重复 ▶

找到答案后，记录 [nums[i], nums[left], nums[right]]。

然后跳过 left 和 right 的重复值（见下面去重详解），最后 left++、right-- 继续搜索。

去重是难点

为什么需要去重？怎么去重？

为什么需要去重？

排序后数组可能有重复元素，比如 [-1, -1, 0, 1, 2]。如果不去重，i=0 和 i=1 都是 -1，会产生相同的三元组。

i 层去重

if (i > 0 && nums[i] === nums[i-1]) continue

跳过和前一个相同的 i，避免固定相同的数

left / right 层去重

找到答案后：

while (left < right && nums[left] === nums[left+1]) left++

while (left < right && nums[right] === nums[right-1]) right--

跳过重复的 left 和 right 值，避免记录相同的三元组

关键前提：排序让相同值相邻，才能通过和相邻元素比较来跳过重复。

完整代码

function threeSum(nums) {
  nums.sort((a, b) => a - b)
  const result = []
  for (let i = 0; i < nums.length - 2; i++) {
    if (nums[i] > 0) break                        // 剪枝：最小值 > 0，不可能三数之和为 0
    if (i > 0 && nums[i] === nums[i - 1]) continue  // i 层去重
    let left = i + 1, right = nums.length - 1
    while (left < right) {
      const sum = nums[i] + nums[left] + nums[right]
      if (sum < 0) {
        left++
      } else if (sum > 0) {
        right--
      } else {
        result.push([nums[i], nums[left], nums[right]])
        while (left < right && nums[left] === nums[left + 1]) left++    // left 去重
        while (left < right && nums[right] === nums[right - 1]) right--  // right 去重
        left++
        right--
      }
    }
  }
  return result
}

空间 O(1) 不算排序空间和返回结果，只用了指针变量
