#3 无重复字符的最长子串 Medium

Hot 100 · 滑动窗口 · 第2周

图示

题目

给定一个字符串 s，请你找出其中不含有重复字符的最长子串的长度。

输入：s = "abcabcbb"
输出：3
解释：无重复字符的最长子串是 "abc"，长度为 3

先想一想

暴力法怎么做？为什么不好？

暴力法：枚举所有子串 O(n²)，每个子串检查是否有重复字符 O(n)。

总时间复杂度 O(n³)，太慢了！

我们需要滑动窗口，把复杂度降到 O(n)。

滑动窗口核心思想

维护一个窗口 [left, right]，right 不断右移扩大窗口；遇到重复字符就 left 右移缩小窗口，直到窗口内无重复。每一步都更新最大长度。

思路（逐步展开）

点击每一步展开详细说明

1 用 Set 记录窗口内的字符 ▶

创建一个 Set，用来存储当前窗口 [left, right] 内的所有字符。

Set 的 has() 和 delete() 都是 O(1)，非常适合做"是否重复"的判断。

2 right 右移，字符加入 Set ▶

用 for 循环让 right 从 0 遍历到 s.length - 1。

每次循环尝试把 s[right] 加入窗口。

3 如果新字符已在 Set 中 → left 右移 ▶

如果 s[right] 已经在 Set 中，说明窗口内有重复了。

此时需要不断把 s[left] 从 Set 中删除，并 left++，直到窗口内没有 s[right] 这个字符。

关键：用 while 循环逐个删除，因为重复的字符不一定在 left 位置，可能在中间。
4 每步更新 maxLen ▶

窗口调整完成后，当前窗口长度为 right - left + 1。

maxLen = Math.max(maxLen, right - left + 1)

例如 "abcabcbb"：

// right=0: set={a}         maxLen=1
// right=1: set={a,b}       maxLen=2
// right=2: set={a,b,c}     maxLen=3
// right=3: s[3]='a'重复! left移到1 → set={b,c,a} maxLen=3
// right=4: s[4]='b'重复! left移到2 → set={c,a,b} maxLen=3
// ... 最终返回 3

完整代码

function lengthOfLongestSubstring(s) {
  const set = new Set()
  let left = 0, maxLen = 0
  for (let right = 0; right < s.length; right++) {
    while (set.has(s[right])) {
      set.delete(s[left])
      left++
    }
    set.add(s[right])
    maxLen = Math.max(maxLen, right - left + 1)
  }
  return maxLen
}

复杂度分析

维度复杂度说明
时间 O(n) left 和 right 各最多移动 n 次，总共 2n
空间 O(min(m, n)) m 为字符集大小，Set 最多存 m 个字符

进阶思考

用 Map 代替 Set 可以更快吗？

可以！用 Map 存 {字符 → 最后出现位置}。

遇到重复字符时，直接把 left 跳到重复位置 + 1，不需要 while 循环逐个删除。

function lengthOfLongestSubstring(s) {
  const map = new Map()
  let left = 0, maxLen = 0
  for (let right = 0; right < s.length; right++) {
    if (map.has(s[right]) && map.get(s[right]) >= left) {
      left = map.get(s[right]) + 1
    }
    map.set(s[right], right)
    maxLen = Math.max(maxLen, right - left + 1)
  }
  return maxLen
}

虽然都是 O(n)，但 Map 版本省去了 while 循环，常数更小。

滑动窗口模板总结

通用模板三步走：
1. right 扩 —— 右指针右移，扩大窗口
2. 条件不满足 left 缩 —— 左指针右移，缩小窗口直到条件满足
3. 每步更新答案 —— 更新最大/最小值

这是所有滑动窗口题的通用模板，记住这三步就能解决大部分滑动窗口问题。
