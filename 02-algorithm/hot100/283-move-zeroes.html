#283 移动零 Easy

Hot 100 · 双指针 · 第1周

学习进度 得分：0 / 3

题目

给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

要求：必须在原地操作，不能拷贝额外数组。

输入：nums = [0, 1, 0, 3, 12]
输出：[1, 3, 12, 0, 0]

先想一想

最简单的思路是什么？

最容易想到的是：新建一个数组，先把非零元素按顺序放进去，再在末尾补零。

但题目要求原地操作，不能用额外数组。

所以我们需要 —— 双指针：一个指针管"该放哪"，一个指针管"去找谁"。

思路（逐步展开）

点击每一步展开详细说明

1 slow 指针：下一个非零元素该放的位置 ▶

slow 从 0 开始，它始终指向"下一个非零元素应该放置的位置"。

slow 左边的所有元素（[0, slow)）都是已经排好的非零元素。

slow 就像一个"安置员"，只负责接收非零元素。
2 fast 指针：遍历数组找非零元素 ▶

fast 从 0 开始，逐一扫描每个元素。

它就像一个"侦察兵"，负责在数组中找到非零元素，然后交给 slow。

3 遇到非零：swap(slow, fast)，slow++ ▶

当 fast 找到一个非零元素时：

  1. 将 nums[slow] 和 nums[fast] 交换
  2. slow++，准备接收下一个非零元素
// 交换操作
[nums[slow], nums[fast]] = [nums[fast], nums[slow]]
slow++
交换保证了非零元素被放到前面，零被"推"到后面。
4 fast 继续前进，直到遍历完整个数组 ▶

无论是否交换，fast 每一轮都 ++。

遇到 0 就跳过（只有 fast++），遇到非零就交换并推进两个指针。

当 fast 到达数组末尾，所有非零元素都已经被交换到前面，剩下的自然是零。

模拟执行过程

输入：[0, 1, 0, 3, 12]

步骤 fast slow nums[fast] 操作 数组状态
初始 - 0 - - [0, 1, 0, 3, 12]
1 0 0 0 跳过（是零） [0, 1, 0, 3, 12]
2 1 0 1 swap(0,1), slow++ [1, 0, 0, 3, 12]
3 2 1 0 跳过（是零） [1, 0, 0, 3, 12]
4 3 1 3 swap(1,3), slow++ [1, 3, 0, 0, 12]
5 4 2 12 swap(2,4), slow++ [1, 3, 12, 0, 0]
最终结果：[1, 3, 12, 0, 0] — 非零元素相对顺序不变，零全部移到末尾。

完整代码

function moveZeroes(nums) {
  let slow = 0
  for (let fast = 0; fast < nums.length; fast++) {
    if (nums[fast] !== 0) {
      [nums[slow], nums[fast]] = [nums[fast], nums[slow]]
      slow++
    }
  }
}

复杂度分析

时间复杂度
O(n)
空间复杂度
O(1)
只用了一次遍历（fast 从头走到尾），没有嵌套循环。slow 和 fast 都只往前走，不回头。额外空间只用了两个指针变量。

套路总结

双指针入门模板
"一个管位置，一个管遍历"
  • slow（管位置） — 指向"结果数组"的下一个空位
  • fast（管遍历） — 负责扫描原数组，寻找符合条件的元素
  • 找到目标元素 → 放到 slow 位置 → slow 前进
  • 同类题：#26 删除有序数组中的重复项、#27 移除元素
