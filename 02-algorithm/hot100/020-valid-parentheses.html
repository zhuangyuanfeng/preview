#20 有效的括号 Easy

Hot 100 · 栈 · 第4周

题目

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s，判断字符串是否有效。

有效字符串需满足：

  • 左括号必须用相同类型的右括号闭合
  • 左括号必须以正确的顺序闭合
输入：s = "()[]{}"    输出：true
输入：s = "(]"         输出：false
输入：s = "([)]"       输出：false
输入：s = "{[]}"       输出：true

先想一想

用什么数据结构最适合匹配括号？

思路（逐步展开）

点击每一步展开详细说明

1 遍历字符串 ▶

逐个读取字符串中的每个字符，判断它是左括号还是右括号。

2 遇到左括号入栈 ▶

如果当前字符是 (、[ 或 {，就把它 push 到栈里。

左括号需要等待后续匹配的右括号，先存起来。

3 遇到右括号查看栈顶是否匹配 ▶

如果当前字符是 )、] 或 }，弹出栈顶元素，检查是否是对应的左括号。

如果不匹配，直接返回 false。

4 最后栈为空则有效 ▶

遍历结束后，如果栈为空说明所有括号都匹配了，返回 true。

如果栈不为空，说明有多余的左括号没被匹配，返回 false。

模拟过程

以 s = "{[]}" 为例，跟着走一遍

字符 操作 栈
{ 左括号，入栈 [ { ]
[ 左括号，入栈 [ {, [ ]
] 右括号，栈顶 [ 匹配 [ { ]
} 右括号，栈顶 { 匹配 [ ]
结束 栈为空 → return true

代码

时间 O(n)，空间 O(n)
function isValid(s) {
  const stack = []
  const map = { ')': '(', ']': '[', '}': '{' }
  for (const c of s) {
    if (!map[c]) {
      stack.push(c)            // 左括号入栈
    } else {
      if (stack.pop() !== map[c]) return false
    }                            // 右括号：弹出栈顶比较
  }
  return stack.length === 0
}

代码解读

map 用右括号做 key 映射到对应的左括号。

如果当前字符 c 不在 map 中（即 !map[c]），说明它是左括号，入栈。

否则它是右括号，弹出栈顶元素和 map[c]（对应的左括号）比较，不匹配就返回 false。

关键提醒

最后必须检查 stack.length === 0，不能只返回 true。

因为 "(((" 这种情况遍历完不会 return false，但栈不为空，说明有多余的左括号没被匹配。

栈是很多算法题的基础数据结构，括号匹配是最经典的栈应用。

掌握了这道题的思路，后面遇到单调栈、最小栈等题目会更容易上手。
