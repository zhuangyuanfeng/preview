#11 盛最多水的容器 Medium

Hot 100 · 双指针 · 第1周

图示

题目

给定 n 个非负整数 a1, a2, ..., an，每个数代表坐标中的一个点 (i, ai)。画 n 条垂直线，找出其中两条线，使得它们与 x 轴共同构成的容器可以盛最多的水。

输入：height = [1, 8, 6, 2, 5, 4, 8, 3, 7]
输出：49
解释：选择下标 1 和 8 的两条线（高度 8 和 7），宽度 = 7
     面积 = min(8, 7) × 7 = 49

先想一想

暴力法是 O(n²)，如何优化到 O(n)？

双指针对撞：左右两端各一个指针，向中间移动。

每次移动较短的那个指针，因为移动较长的一侧面积只会变小或不变，没有意义。

这样只需要遍历一遍，时间复杂度 O(n)。

核心问题

为什么要移动较短的指针？

双指针解法（逐步展开）

点击每一步展开详细说明

1 初始化双指针 ▶

left = 0（最左端），right = height.length - 1（最右端）。

初始宽度最大，之后每移动一步宽度减 1。

2 计算当前面积 ▶

面积 = 宽 × 高 = (right - left) × Math.min(height[left], height[right])

每次都和 maxWater 取较大值。

3 移动较短指针 ▶

如果 height[left] < height[right]，移动 left++；否则 right--。

关键证明：移动较长指针 → 宽减小，高 = min(两边) 不变或变小 → 面积一定变小。
移动较短指针 → 宽减小，但高可能变大 → 面积有可能变大。
4 模拟执行过程 ▶

height = [1, 8, 6, 2, 5, 4, 8, 3, 7]

L=0 R=8 h=[1,...,7] min(1,7)×8 = 8 移动L (1<7)
L=1 R=8 h=[8,...,7] min(8,7)×7 = 49 移动R (8>7)
L=1 R=7 h=[8,...,3] min(8,3)×6 = 18 移动R (8>3)
L=1 R=6 h=[8,...,8] min(8,8)×5 = 40 移动R (相等)
... 继续直到 L >= R，最终 maxWater = 49
5 完整代码 ▶ 
function maxArea(height) {
  let left = 0, right = height.length - 1
  let maxWater = 0
  while (left < right) {
    const w = right - left
    const h = Math.min(height[left], height[right])
    maxWater = Math.max(maxWater, w * h)
    if (height[left] < height[right]) {
      left++
    } else {
      right--
    }
  }
  return maxWater
}
