#19 删除链表的倒数第N个结点 Medium

Hot 100 · 链表 · 第3周

题目

给你一个链表，删除链表的倒数第n个结点，并且返回链表的头结点。

输入：head = [1,2,3,4,5], n = 2
输出：[1,2,3,5]

先想一想

怎么一次遍历就找到倒数第N个节点？

思路（逐步展开）

点击每一步展开详细说明

1 创建dummy节点 ▶

const dummy = { next: head }

dummy 节点是关键，用来处理删除头节点的边界情况。fast 和 slow 都从 dummy 出发。

2 fast先走n+1步 ▶

for (let i = 0; i <= n; i++) fast = fast.next

为什么是 n+1 步而不是 n 步？因为 slow 需要停在要删除节点的前一个位置，这样才能执行 slow.next = slow.next.next。

3 fast和slow同时走直到fast到null ▶

while (fast) { fast = fast.next; slow = slow.next }

两个指针同速前进，保持间隔不变。当 fast 到达 null 时，slow 正好在目标节点的前一个。

4 删除目标节点 ▶

slow.next = slow.next.next

跳过目标节点，直接指向下下个节点，完成删除。最后返回 dummy.next。

模拟过程

以 [1,2,3,4,5] n=2 为例，跟着走一遍

阶段 fast slow 说明
初始 dummy dummy dummy→1→2→3→4→5→null
fast走3步后 3 dummy fast先走n+1=3步
同时走1步 4 1 间隔保持3
同时走2步 5 2 间隔保持3
同时走3步 null 3 fast到null，停止
删除 - 3 slow.next = slow.next.next → 跳过4 → [1,2,3,5]

代码

时间 O(n)，空间 O(1) —— 一次遍历，双指针
function removeNthFromEnd(head, n) {
  const dummy = { next: head }
  let fast = dummy, slow = dummy
  for (let i = 0; i <= n; i++) fast = fast.next  // fast先走n+1步
  while (fast) {
    fast = fast.next   // 同时走
    slow = slow.next   // 同时走
  }
  slow.next = slow.next.next  // 删除目标节点
  return dummy.next
}

关键提醒

dummy节点是关键！

如果要删除的是头节点（比如 [1], n=1），没有 dummy 会很难处理。加了 dummy 之后，删除头节点和删除其他节点的逻辑完全一样。

fast要先走n+1步不是n步！

因为 slow 需要停在要删除节点的前一个位置，才能执行 slow.next = slow.next.next。如果只走 n 步，slow 会停在目标节点本身，无法删除。
